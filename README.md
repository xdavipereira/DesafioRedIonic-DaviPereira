# DesafioRedIonic- Davi Pereira

Desafio para a seleção de estagiario, da empresa RedRocket.

##Instalação

```
https://github.com/xdavipereira/DesafioRedIonic-DaviPereira.git
cd DesafioRedIonic-DaviPerereira
npm install
```
##Iniciar aplicação.

```
ionic serve --lab
```
